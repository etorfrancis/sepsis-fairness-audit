# Fairness Audit & Debiasing of Sepsis Mortality Prediction Models

**Fairness Audit and Debiasing of Machine Learning Models for Sepsis Mortality
Prediction Across Demographic Subgroups: A Multi-Metric Study**

---

## Overview

This repository contains the complete, reproducible code for a publishable
research study examining algorithmic fairness in ICU sepsis mortality prediction.
The study trains three ML models on a MIMIC-IV-calibrated cohort, audits them
across race, sex, and insurance status using four complementary fairness metrics,
applies equalized-odds post-processing debiasing, and quantifies the
accuracy–fairness Pareto tradeoff — the paper's core novel contribution.

## Quickstart

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the full pipeline
cd sepsis_fairness_audit
python src/main.py
```

The pipeline will:
1. Generate (or load) the `sepsis_cohort.csv` dataset
2. Preprocess: MICE imputation, stratified 70/15/15 split, RobustScaler
3. Train: Logistic Regression, XGBoost, MLP
4. Audit: 4 fairness metrics × 3 protected attributes × 3 models
5. Debias: per-group threshold optimisation (Hardt et al. 2016)
6. SHAP: global importance + Fairness-SHAP attribution (Black vs White)
7. Figures: 8 publication-ready PDFs saved to `figures/`

Runtime: ~3–5 minutes on a modern CPU.

## Using Real MIMIC-IV Data

This study was designed and validated on a synthetic cohort calibrated to
published MIMIC-IV sepsis statistics. To run on real MIMIC-IV data:

1. Complete CITI training at https://citiprogram.org
2. Apply for PhysioNet access at https://physionet.org/register
3. Extract the sepsis cohort using Sepsis-3 criteria (SOFA ≥ 2 + suspected
   infection) from MIMIC-IV hosp + icu modules
4. Save as `sepsis_cohort.csv` with the same column schema (see `preprocessing.py`)
5. Run `python src/main.py` — it will load your file automatically

The `preprocessing.py` module includes schema validation and clinical range
assertions that will flag any extraction issues before training begins.

## Study Design Summary

### Research Question
Do ML-based sepsis mortality prediction models exhibit demographic disparities,
and can post-processing debiasing reduce these disparities without unacceptable
accuracy loss?

### Models
| Model | Notes |
|-------|-------|
| Logistic Regression | Interpretable baseline, L2 regularised |
| XGBoost | State-of-the-art tabular, early stopping |
| MLP | Shallow 2-layer neural network |

### Fairness Metrics (per model × protected attribute × subgroup)
| Metric | Definition | Clinical interpretation |
|--------|------------|------------------------|
| **Equalized Odds Difference (EOD)** | max(|ΔTPR|, |ΔFPR|) | Are we missing deaths at the same rate? |
| **Demographic Parity Difference (DPD)** | |Δ positive prediction rate| | Are alarm rates equal? |
| **Predictive Parity (PPV gap)** | |Δ precision| | When alarm fires, equally reliable? |
| **Subgroup ECE** | Calibration error per group | Do probabilities reflect true risk? |

### Protected Attributes
- Race/Ethnicity (White, Black, Hispanic, Asian, Other)
- Sex (Male, Female)
- Insurance type (Medicare, Medicaid, Private, Self-pay)

### Debiasing
- Method: Per-group TPR-equalising threshold optimisation (Hardt et al., NeurIPS 2016)
- Evaluation: Accuracy–Fairness Pareto frontier (novel contribution)

### Novel Contribution
Most sepsis ML papers report only overall AUROC. This study:
1. Applies **4 complementary fairness metrics** (not just AUROC by race)
2. Performs **formal debiasing** with quantified accuracy cost
3. Produces the **accuracy–fairness Pareto frontier** — enabling clinicians to
   choose their operating point on the tradeoff curve
4. Extends SHAP to **Fairness-SHAP attribution** — which features drive
   between-group prediction disparities

---

## Key Findings (Synthetic Cohort)

- **Black patients had the highest mortality** (57.2%) vs White (46.6%), consistent
  with documented ICU racial disparities
- **LR showed the worst racial bias**: EOD = 0.1925 for Black patients
- **XGBoost was most equitable** overall despite strong performance
- **time_to_antibiotics_h** showed the largest Black–White SHAP gap, consistent
  with documented care delivery disparities
- **Debiasing reduced EOD by 6.1% (LR) with no AUROC change** — demonstrating
  a favourable tradeoff point on the Pareto frontier
